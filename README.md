Ui del proyecto
Este repositorio contiene exclusivamente la capa de presentación por consola del proyecto. Incluye los menús de interacción (MenuAutenticacion, MenuPrincipal, MenuUsuario, MenuDepartamento, MenuTicket, MenuDiccionario), la clase IO para el manejo de entradas mediante BufferedReader, y la clase Controller, que actúa como puente entre la UI y la lógica importada desde el .jar generado en el repositorio de negocio. Esta capa no contiene reglas del dominio ni almacenamiento; su responsabilidad es mostrar opciones, recibir datos del usuario y delegar todas las operaciones a la lógica central.
