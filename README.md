Este repositorio contiene exclusivamente la capa de presentación por consola del sistema HelpDesk U. Su función es gestionar la interacción directa con la persona usuaria mediante menús textuales, sin incluir lógica de negocio ni acceso directo a datos. La UI se compone de los menús MenuAutenticacion, MenuPrincipal, MenuUsuario, MenuDepartamento, MenuTicket y MenuDiccionario, cada uno encargado de presentar opciones específicas y capturar la información ingresada por el usuario.
